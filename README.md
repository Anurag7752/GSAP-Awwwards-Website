Tech Stack
⚛️ React 19

Quick Start
# 1. Clone the repo
git clone [https://github.com/FullStackEmpire/gsap-awwwards-website.git](https://github.com/FullStackEmpire/gsap-awwwards-website.git)

# 2. Install dependencies
npm install
# or
yarn

# 3. Start the dev server
npm run dev

yarn dev
